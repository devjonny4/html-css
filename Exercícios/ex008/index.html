Principais formatações

Negrito/Destaque

Nessa frase, temos um termo em negrito usando a tag <b> (não semântica).

Nessa frase, tenho um termo em destaque usando a tag <strong> (semântica)

Itálico/Ênfase

Nessa frase, temos um termo em itálico usando a tag <i> (não semântica)

Nessa frase, temos um termo em ênfase usando a tag <em> (semântica)

Texto marcado

Podemos criar também um texto marcado usando a tag <mark>.

Texto grande e pequeno

Estamos criando um texto grande e um texto pequeno esse paragrafo usando as tags <big> e <small>
(a tag big está em desuso por ser apenas uma tag de forma).

Texto deletado e texto inserido

Podemos marcar um texo como excluído para indicar que ele deve ser lido, mas não considerado, usando a tag <del>.

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois, usando a tag <ins>

Texto sobrescrito e subscrito

Para inserir coisas do tipo x20+3 (tag <sup>)

Para inserir coisas do tipo H2O (tag <sub>)
